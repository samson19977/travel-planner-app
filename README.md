# 🤖 TRAVEL WITH ME - AI Travel Planner

An intelligent multi-agent travel planning system that generates personalized itineraries using real-time data from multiple APIs.

## 🌟 Features

- **Weather Agent** - Real-time weather data for any destination
- **Scout Agent** - Discovers top attractions using AI
- **Budget Agent** - Smart budget optimization based on your spending
- **Planner Agent** - Creates day-by-day itineraries with max 3 activities per day

## 🛠️ Technologies Used

- **Backend**: FastAPI (Python)
- **Frontend**: HTML/CSS/JavaScript
- **APIs**: 
  - OpenAI GPT-3.5 for attractions
  - OpenWeatherMap for weather data
  - SerpAPI for destination images
- **Deployment**: Vercel

## 📋 Prerequisites

- Python 3.8+
- API Keys (see Environment Variables section)
